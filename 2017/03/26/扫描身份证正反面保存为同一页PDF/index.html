扫描身份证正反面保存为同一页PDF

发表于 2017-03-26   |     |     1

最近扫描证件存档时遇到一个问题——身份证正反面要在同一页上。设想了几种方案：

  • 利用pdf阅读器的编辑功能，将2页pdf合成一页：

    搜索关键词merge pdf to one page，有一些网页版的可以实现，但是合成的一页特别长，是2张A4的长度，即使我提前在2页上各截出一半。更不用说，这些软件生成的新pdf都带有很大的水印了。

  • 扫描时保存为2张图片，然后利用万能ps软件，合成一张A4大小。但是有一个问题，我不知道图片要调整到多大的尺寸才能与实际相符，也没有合适的测量工具。

  • 最后我的实现方法是，在pdf里插入图片实现：

    一般阅读器是没有这个功能的，不过曲线救国的方法是利用自定义图章实现。

    先扫描正面保存为pdf文件，然后将扫描保存的正反面图片设定成自定义的2个图章，在同一页上插入这2个图章，调整它们的大小刚好覆盖住原pdf上的大小。大功告成！

    ​

后记 ：也许是pdf格式为了保护版权的专有限制吧，也许是需求的人太少，在2页上分别截取部分，在重新合成一页的功能怎么就没有呢！
